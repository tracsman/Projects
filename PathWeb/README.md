# PathWeb

## Overview
ASP.NET Core 8 MVC application with Microsoft Entra ID authentication and role-based authorization.

## Features
- **Entra ID Authentication**: Integrated Microsoft Entra ID for user authentication
- **Role-based Authorization**: Local authorization support ready for implementation
- **Entity Framework Core**: EF Core 8 with SQL Server provider
- **MVC Areas**: Four separate areas for modular organization

## Prerequisites
- .NET 8.0 SDK
- Microsoft Entra ID tenant
- SQL Server (when ready to add database)

## Configuration
Update appsettings.json with your Entra ID credentials before running.
